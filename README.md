# TIPE22-23

Projet effectué dans le cadre d'une présentation orale pour les concours d'écoles d'ingénieur en 2023.
Ce projet porte sur l'étude du fonctionnement des moteurs thermiques et l'optimisation de ces derniers face aux enjeux environnementaux.
Dans l'optique d'une transition plus fluide vers les véhicules à moteur électrique, on propose de limiter l'usage des moteurs à explosion aux heures d'affluence et de proposer un itinéraire en accord avec les enjeux écologiques pour un véhicule se déplaçant à travers une ville.
